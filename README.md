# LeafNote

`LeafNote.html` は、単一 HTML ファイルで動作するローカル向けブロック型エディタです。
インストールやビルドなしで、ブラウザで直接開いて利用できます。

## 起動方法
1. LPを見る場合は `index.html` をブラウザで開きます。
2. Swiss editorial design 版のLPを見る場合は `index2.html` をブラウザで開きます。
3. エディタを直接使う場合は `LeafNote.html` をブラウザで開きます。
4. そのまま編集を開始できます。

## Markdown Native 実験版
- `LeafNote.html` は Markdown Native 版として、Markdown 設計書向けの表現を扱いやすくしています。
- Markdown 設計書向けに、目次と Mermaid を追加しています。
- Mermaid はコードフェンス付き/なしで、Mermaid 公式 Syntax Reference 掲載の図種宣言を Mermaid ブロックとして取り込みます。フローチャート、シーケンス図、ER図、ガントチャート、状態遷移図は簡易SVGプレビューを表示し、それ以外の公式図種は図種名付きの Mermaid ソースプレビューとして保持します。
- Decision、Risk、Raw Markdown など必要性が薄い追加ブロックはメニューから外し、既存データは通常テキストまたはコールアウトへ移行します。
- HTML は互換保持用の内部ブロックとして取り込み、未対応 Markdown は Raw Markdown ではなく通常テキストとして取り込みます。
- 保存先は Markdown Native 版専用の IndexedDB / localStorage キーを使います。

## 主な機能
- サイドバーでノートの作成・複製・階層管理
- ノートのゴミ箱移動・復元・完全削除
- ブロック編集（見出し、箇条書き、番号付き、ToDo、引用、コールアウト、コード、区切り線、テーブル、画像）
- ブロック左の `+` ボタン、`/` スラッシュコマンド、Markdown ショートカットによるブロック挿入・変換
- 検索モーダル（`Cmd/Ctrl + K`）
- ライト/ダーク切替とテーマカスタマイズ（テーマカラー、フォント、ページ幅）
- Markdown インポート / エクスポート
- アプリ全体を 1 ファイルの HTML として書き出し
- 複数ブロックの範囲選択、Markdown コピー / カット / 貼り付け、Undo / Redo

## UI と操作性
- favicon は黒背景を透過したPNGをbase64化してHTML内に埋め込んでおり、外部ファイルなしで表示されます。
- 各ノートのアイコンはエディタ見出し、サイドバー、検索結果、ゴミ箱に反映されます。
- アイコンピッカーは、ノート・タスク・仕事・生活で使いやすい絵文字と Material アイコンだけに絞っています。
- サイドバーの子ノート展開アイコンは、表示可能な子ノートがある場合だけ展開状態を示します。
- サイドバー、検索結果、コンテキストメニュー、ショートカット一覧などのクリック項目はキーボード操作しやすい要素に調整しています。
- 半幅・タブレット幅ではサイドバーを通常の横並びで表示し、ノート本文を隠さないようにしています。
- スマートフォン幅ではサイドバーを初期表示で折りたたみ、開いたときはドロワーとして表示します。
- 小さい画面ではブロック操作、コールアウト/引用、コードブロック、画像追加、検索・設定・ゴミ箱・ショートカットの各モーダルが画面内に収まるように調整しています。テーブルは列幅を保ち、ブロック内で横スクロールします。
- ダークモードではコールアウトなどの強調背景に専用の濃淡と枠線を使い、本文背景から判別しやすくしています。
- LP（`index.html`）は LeafNote のソース配布導線として作成しており、生成した葉の背景と LeafNote ワードマーク、短いコピー、`Copy SourceCode`、GitHubリポジトリへのアイコン導線だけで構成した Swiss editorial design 風の単一HTMLとして外部CDNなしで表示します。`Copy SourceCode` は `index.html` 内に埋め込んだ `LeafNote.html` のソースをそのままクリップボードへコピーし、完了時は中央の大きなトーストで知らせます。
- LP（`index2.html`）は思想を伝える別案として作成しており、短い英語コピー、大きな余白、PNGの紙片・スキャン質感・手書きチェック・HTMLラベル風アクセントだけで構成しています。
- モーダルは `role="dialog"` とフォーカス復元を備え、`Escape` で閉じられます。
- ホバーで出る操作部品は、キーボードフォーカス時やタッチ環境でも見えるようにしています。
- テーマカスタマイズのテーマカラーは左端と初期値を緑系にし、それ以外は青・シアン・橙・赤・紫・グレー系から選べます。選択したテーマカラーは、選択範囲、フォーカス、コールアウト、表、コード、画像追加枠などにも反映されます。
- フォーカス中の空のテキスト、見出し、リスト、ToDo、引用、コールアウト、コードブロックにはプレースホルダーを表示します。
- `prefers-reduced-motion` が有効な環境では、不要なアニメーションを抑制します。

## 保存仕様
- メイン保存先は `IndexedDB`（自動保存）です。
- `IndexedDB` 保存に失敗した場合は `localStorage`（キー: `leafnote-markdown-native-v1`）に退避し、画面右下に警告を表示します。
- `IndexedDB` と `localStorage` の両方に保存できない場合は、画面右下にエラーを表示します。
- ページを閉じる直前や非表示化時（`pagehide` / `beforeunload` / `visibilitychange`）は、保留中の変更を即時退避して取りこぼしを減らします。
- ノート名とドキュメント名はプレーンテキストとして保存し、`<` / `>` / `&` などの文字をリロード後もそのまま保持します。
- HTML 書き出し時は、現在状態を `embedded-state` に埋め込みます。
- 書き出し済み HTML を同じブラウザで開いた場合、埋め込み状態よりブラウザ内の自動保存状態が新しければ、新しい方を優先します。
- HTML 書き出し時に `http/https` 画像URLは可能な限り `data:` URL 化し、自己完結ファイルとして再現性を高めます（変換失敗時は警告を表示）。

## ゴミ箱と自動削除
- ノート削除はまずゴミ箱への移動（ソフト削除）です。
- 保持日数（0 / 7 / 30 / 90日）を設定すると、期限切れノートは自動で完全削除されます。
- 自動削除が実行された場合は件数通知を表示し、ゴミ箱モーダル内にも直近実行履歴を表示します。
- 完全削除は元に戻せません。

## 主要ショートカット
- `Enter`: 次ブロック作成
- `Shift + Enter`: ブロック内改行
- `Backspace`（空行）: ブロック削除
- `Cmd/Ctrl + K`: 検索モーダル
- `Cmd/Ctrl + Z`: 元に戻す
- `Cmd/Ctrl + Shift + Z` または `Cmd/Ctrl + Y`: やり直し

## 既知の注意点
- コードブロックは外部 CDN に依存しないプレーン表示です。Prism が別途読み込まれている環境ではハイライトに対応します。
- ブラウザのプライベートモードやストレージ容量制限により、保存が失敗する場合があります。
